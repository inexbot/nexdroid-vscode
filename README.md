
# NexDroid

## 简介

这个插件是用来辅助在VSCode中手动编写纳博特控制系统的程序的。

## 使用

当前本插件有两个功能，代码自动补全、说明与代码高亮。

本插件涉及到的指令请参照我们的[指令说明书](https://www.inexbot.com/manual/)

**在编写程序时请确保已经充分了解本系统，以及所写指令功能。**
